## Overview
Book-Store is a fully functional Django-based e-commerce platform for online book shopping.  
It includes **user authentication, shopping cart, wishlist, checkout, payment integration**, and **admin management**, along with **customer support features** like live chat and WhatsApp integration.  

This project demonstrates a complete e-commerce workflow using Django, ideal for learning and showcasing full-stack development skills.

---

## Key Features

### User Features
- Attractive home page with featured books and collections  
- Browse books by category or collection  
- Product detail pages with images and descriptions  
- Shopping cart & wishlist management  
- Checkout and order placement with order summary  
- Secure payment using Razorpay  
- User registration, login, and profile management  
- Media uploads for products  
- Live chat & WhatsApp integration for customer support  

### Admin Features
- Admin panel to manage products, collections, and orders  
- Track, approve, and update orders  
- Add, edit, and remove books, manage inventory  
- Monitor user accounts and activity  

### Additional Features
- Responsive design for desktop and mobile  
- Clean and modular Django project structure  
- Optimized for fast browsing and smooth user experience  

---

## Technology Stack
- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite  
- **Payment Gateway:** Razorpay  
- **Extras:** Live chat & WhatsApp integration
